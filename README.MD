# Healthcare Workflow System

A full-stack Healthcare Workflow Management System built using React.js, Spring Boot, PostgreSQL, Hibernate, and REST APIs.

---

## Project Overview

This application helps manage patient healthcare workflows for hospitals and clinics.

The system allows:

* Viewing patient records
* Adding new patients
* Updating patient details
* Deleting patient records
* Backend API integration
* Database management using PostgreSQL
